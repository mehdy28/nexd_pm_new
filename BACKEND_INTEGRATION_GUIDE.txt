# NEXD.PM Backend Integration Guide

## Overview
This guide explains how to connect the GraphQL backend to your NEXD.PM frontend and remove mock data.

## Prerequisites
1. PostgreSQL database (local or cloud)
2. Node.js 18+ installed
3. npm or yarn package manager

## Step 1: Install Dependencies
Add these packages to your package.json:

```bash
npm install @apollo/server @apollo/client graphql @graphql-tools/schema
npm install @prisma/client prisma
npm install next-auth @next-auth/prisma-adapter
npm install bcryptjs jsonwebtoken
npm install @types/bcryptjs @types/jsonwebtoken --save-dev
.
